## 🚀 使い方

### 1. ローカルで起動
```bash
# ファイルをダウンロード後、ブラウザで index.html を開く
open index.html
# または
python -m http.server 8000  # Python 3
# http://localhost:8000 でアクセス
```

### 2. 機能の使い方
- **ナビゲーション**: ヘッダーのリンクをクリックで各セクションへジャンプ

- **印刷**: ヒーローセクションの「PDFで印刷」ボタンまたはブラウザの印刷機能

## 🔧 カスタマイズ方法

### 1. カラー変更
`styles.css` の `:root` セクションでカスタムプロパティを編集：
```css
:root {
  --brand: #5aa9e6;  /* メインカラー */
  --accent: #82e0aa; /* アクセントカラー */
}
```

### 2. フォント変更
`styles.css` の `--font-family` を変更：
```css
--font-family: "Your Font", sans-serif;
```

### 3. コンテンツ更新
`index.html` の各セクションを直接編集（提供データを厳守）

### 4. メールアドレス更新
`index.html` の `mailto:` リンクを実際のアドレスに変更：
```html
<a href="mailto:your-email@example.com" class="btn btn--primary">
```
